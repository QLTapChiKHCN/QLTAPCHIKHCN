Cấu hình dự án laravel kết hợp với SQL sever
1. lấy 2 file trong git để vào etx trong thư mục php trong xampp
2. mở php.ini gắn 2 cái extension= tên file ở trên ( lưu ý là 2 cái extension nha)
3. trong file cấu hình env gắn cụm này vào 
DB_CONNECTION=sqlsrv
DB_HOST=127.0.0.1
DB_PORT=1433 // cổng mặc định
DB_DATABASE=QL_TCKHCN // tên db
DB_USERNAME=sa //ten đăng nhập 
DB_PASSWORD=123 // mật khẩu trong db
4. mở thư mục config chọn database tìm đến dòng có cái này
'sqlsrv' => [
            'driver' => 'sqlsrv',
            'url' => env('DATABASE_URL'),
            'host' => env('DB_HOST', 'localhost'),
            'port' => env('DB_PORT', '1433'),
            'database' => env('DB_DATABASE', 'QL_TCKHCN'),// sửa lại chỗ này nếu sai tên database
            'username' => env('DB_USERNAME', 'sa'),// chỗ này
            'password' => env('DB_PASSWORD', '123'),// chỗ này 
            'charset' => 'utf8',
            'prefix' => '',
            'prefix_indexes' => true,
            // 'encrypt' => env('DB_ENCRYPT', 'yes'),
            // 'trust_server_certificate' => env('DB_TRUST_SERVER_CERTIFICATE', 'false'),
        ],
còn kiểm tra nếu đúng r thì để nguyên.
